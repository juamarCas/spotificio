# spotificio
This is a simple app that shows you the artist you follow on Spotify,
and show you new releases in a given time

## Config
Create a .env file like this way and fill the parameters
```
CLIENT_ID = 
CLIENT_SECRET =
SPOTIFY_TOKEN = 
PORT = 
CALLBACK_URL = 
```
